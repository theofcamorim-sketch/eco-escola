# EcoEscola

Aplicativo Android desenvolvido para monitoramento e incentivo à sustentabilidade escolar.

## Funcionalidades
- Registro de ações de reciclagem
- Controle de consumo de energia e água
- Gestão de horta escolar
- Sistema de pontuação gamificado

## Tecnologias
- Kotlin
- Firebase
- Figma (prototipagem)
